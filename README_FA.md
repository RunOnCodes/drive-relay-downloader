# 📥 دانلود کننده رله Google Drive

ابزار رابط کاربری دسکتاپ که هر فایلی (از هر URL عمومی) را دانلود کرده و مستقیماً در **Google Drive** شما ذخیره می‌کند – با استفاده از Google Apps Script به عنوان رله.

## ✨ ویژگی‌ها

* **دور زدن ذخیره‌سازی محلی:** فایل مستقیماً از URL منبع به Google Drive شما می‌رود. هرگز به درایو سخت محلی شما دانلود نمی‌شود.
* **رابط کاربری ساده:** یک رابط دسکتاپ آسان‌الاستفاده که با Tkinter ساخته شده است.
* **استخراج خودکار:** نام فایل را به‌طور خودکار از URL استخراج می‌کند اگر نام سفارشی ارائه نشده باشد.
* **پوشه‌های سفارشی:** به‌صورت اختیاری شناسه‌های پوشه Google Drive متفاوت را مستقیماً در رابط کاربری مشخص کنید تا دانلودهای خود را در حین انجام مرتب کنید.
* **راه‌اندازی خاموش:** شامل یک محفظه VBScript برای کاربران Windows برای راه‌اندازی برنامه بدون پنجره دستور دائمی.

---

## 🚀 نحوه کار

1.  شما یک **URL فایل** فراهم می‌کنید (به‌عنوان‌مثال، `https://example.com/file.pdf`).
2.  رابط کاربری Python آن URL و نام فایل دلخواه را به **Google Apps Script وب برنامه** می‌فرستد.
3.  Apps Script ورودی‌ها را تأیید کرده، فایل راه‌دور را دریافت می‌کند، و آن را در **پوشه Google Drive** ذخیره می‌کند.
4.  اسکریپت یک پاسخ موفقیت‌آمیز و URL فایل Drive جدید را برمی‌گرداند و رابط کاربری آن را نمایش می‌دهد.

## 🛠️ دستورالعمل‌های راه‌اندازی

### 1. استقرار Google Apps Script

* [script.google.com](https://script.google.com) را باز کنید و یک پروژه جدید ایجاد کنید.
* کد پیش‌فرض را با محتوای [`ScriptToDeploy.gs`](ScriptToDeploy.gs) جایگزین کنید.
* **شناسه پوشه را تغییر دهید** در داخل اسکریپت:  
    `const folderId = 'YOUR_DRIVE_FOLDER_ID';`  
    *(شناسه پوشه خود را در URL Drive آن بیابید: `https://drive.google.com/drive/folders/THIS_IS_THE_ID`)*
* به‌عنوان **وب برنامه** استقرار دهید:
    * اجرا شود توسط: **من**
    * کی دسترسی دارد: **هرکسی** (یا "هر کسی با لینک" – ابزار باید بتواند به آن دسترسی داشته باشد).
    * روی **استقرار** کلیک کنید → **استقرار جدید** → **وب برنامه**.  
    * **URL استقرار** را کپی کنید (شبیه `https://script.google.com/macros/s/.../exec` به نظر می‌رسد).

### 2. محیط Python را پیکربندی کنید

* مطمئن شوید که **Python 3.7+** نصب شده است.
* کتابخانه `requests` مورد نیاز را نصب کنید:  
    ```bash
    pip install requests
    ```
* `relay_config.json` را ویرایش کنید و URL Apps Script خود را جایگزین کنید:  
    ```json
    {
      "script_url": "https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec",
      "folder_id": ""
    }
    ```
    *(متناوباً، می‌توانید URL را مستقیماً در رابط کاربری وارد کنید و روی "ذخیره تنظیمات" کلیک کنید).*

## 💻 استفاده

### راه‌اندازی برنامه

شما دو روش برای شروع برنامه دارید:
1.  **راه‌اندازی استاندارد:** اسکریپت Python را مستقیماً از ترمینال یا دستور شما اجرا کنید:
    ```bash
    python drive_relay_gui.py
    ```
2.  **راه‌اندازی خاموش (فقط Windows):** روی [`launch_gui.vbs`](launch_gui.vbs) دوبار کلیک کنید. این اسکریپت از `pythonw` استفاده می‌کند تا رابط گرافیکی را بدون نگه‌داشتن پنجره دستور به‌طور خاموش در پس‌زمینه اجرا کند.

### استفاده از رابط کاربری

* **URL Apps Script:** URL برنامه وب را که در مرحله 1 تولید کردید جایگزین کنید.
* **شناسه پوشه Drive (اختیاری):** اگر می‌خواهید یک فایل خاص را به پوشه‌ای متفاوت از پوشه پیش‌فرض‌ای که در Apps Script خود کدگذاری کرده‌اید ذخیره کنید، شناسه پوشه جدید را اینجا جایگزین کنید.
* **URL فایل:** لینک دانلود مستقیم به فایل.
* **ذخیره به‌عنوان (اختیاری):** یک نام سفارشی فراهم کنید (به‌عنوان‌مثال، `report_2026.pdf`). اگر خالی باقی مانده باشد، برنامه به‌طور خودکار نام را از URL تشخیص می‌دهد.
* روی **"📥 دانلود به Drive"** کلیک کنید.

## ⚠️ محدودیت‌ها و یادداشت‌ها

* **حد 50 مگابایت فایل:** Google Apps Script یک حد حافظه سختی برای دریافت و دستکاری بلوب‌ها دارد (`UrlFetchApp`). رابط کاربری Python شما را هشدار می‌دهد اگر فایلی بزرگ‌تر از 50 مگابایت تشخیص دهد، زیرا احتمال اینکه Apps Script از حافظه فرار کند زیاد است.
* **تایم‌آوت:** حد اجرای Apps Script 6 دقیقه است. سرورهای منبع بسیار کند ممکن است باعث شوند که اسکریپت قبل از اتمام انتقال فایل به تایم‌آوت برسد.
* **فقط لینک‌های مستقیم:** URL ارائه‌شده باید یک لینک فایل مستقیم باشد (باید بلافاصله یک دانلود را هنگام جایگزین کردن در مرورگر شروع کند) و عمومی‌تر قابل دسترسی باشد. صفحاتی که نیاز به ورود یا کلیک کردن دارند کار نخواهند کرد.
